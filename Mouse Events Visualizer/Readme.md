# Mouse Event Color Changer

A beginner-friendly JavaScript project that demonstrates different DOM mouse events by changing the webpage's background color.

## Features

* Change the background color using different mouse events.
* Demonstrates the use of:

  * `click`
  * `dblclick`
  * `mouseup`
  * `mouseenter`
  * `mouseleave`

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## How to Run

1. Clone or download the project.
2. Open `index.html` in your browser.
3. Interact with the buttons using the specified mouse events to see the background color change.

## Learning Outcomes

* Selecting HTML elements using the DOM.
* Handling different mouse events with `addEventListener()`.
* Dynamically updating CSS using JavaScript.
