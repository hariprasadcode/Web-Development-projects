# 🧮 Dynamic Calculator

A modern, responsive, and interactive calculator built using **HTML**, **CSS**, and **JavaScript**. This project performs real-time mathematical calculations with a clean user interface, smooth interactions, and essential calculator operations.

---

## 📌 Overview

The **Dynamic Calculator** is a lightweight web application designed to provide a fast and intuitive calculation experience. It mimics the behavior of a real calculator by allowing users to enter expressions dynamically and evaluate them instantly.

This project was built to strengthen core JavaScript concepts such as **DOM Manipulation**, **Event Handling**, **Expression Evaluation**, and **User Interface Design**.

## 🚀 How It Works

1. Click the number buttons to enter values.
2. Choose an arithmetic operator.
3. Continue entering numbers.
4. Press **=** to evaluate the expression.
5. Use **DEL** to remove the last character.
6. Use **AC** to clear the display completely.
