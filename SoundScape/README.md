## 📖 About

**SoundScape** is a responsive music preview application built using **HTML**, **CSS**, and **JavaScript**. It fetches song data from the **iTunes Search API** and displays album artwork, song information, artist details, and a playable audio preview.

This project demonstrates how to work with **REST APIs**, **Fetch API**, **DOM Manipulation**, and **Dynamic Content Rendering** in JavaScript.

---

## ✨ Features

- 🎵 Browse songs from the iTunes API
- 🖼️ Album artwork display
- 🎤 Artist information
- 💿 Album details
- ▶️ 30-second audio preview
- 📱 Responsive card layout
- ⚡ Fast loading using Fetch API
- 🎨 Clean and modern UI
